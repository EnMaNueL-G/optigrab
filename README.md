# OptiGrab

Reproductor y **descargador privado** de vídeo y audio: YouTube, TikTok, Instagram, X/Twitter, Facebook, Vimeo y **+1000 sitios**. Para **Windows y Android**. Gratis, sin anuncios ni rastreadores — todo se procesa en tu equipo.

## Descargas
- **Windows** (portable) y **Android** (APK): pestaña **Releases** de este repositorio, o en **https://optisuite.app**

El **código fuente es privado**. Parte del ecosistema **OptiSuite** · https://github.com/EnMaNueL-G
